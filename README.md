# ElevateX-Flight-Landing-Analyzer
This Tampermonkey script lets you analyze yours and other peoples landings on the ElevateX online map

It displays:
- Flightnumber
- Aircraft
- Route
- Pilot
- All landings
- - FPM
  - G Forces
  - Bounces
  - Ground Speed
  - Wind info (direction speed)
  - Aicraft heading/bank/pitch

1. Install Tampermonkey in the browser of your choice
2. Create a new script, insert the code from the "ElevateX Flight Landing Analyzer-2.0.user.js" file
3. Save and enable

Either
4. Go to "https://map.elevatex.app/"
6. Click on a flight thats already landed
7. Click "Analyze current flight" 

4. Open your flight URL (something like) "https://map.elevatex.app/XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX"
5. Click "Analyze current flight"
